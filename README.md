# Career Guidance System

A web-based assessment tool that provides personalized career recommendations based on user interests, strengths, and work preferences.

## Overview

The Career Guidance System is an interactive web application that helps users explore career options through a structured assessment process. The platform generates career recommendations by analyzing user input data against a comprehensive career database.

## Features

- Four-step assessment process with name, interest, strength, and work style selection
- Intelligent strength suggestions based on selected interest area
- Career compatibility scoring and ranking
- Export results in text or PDF format
- Data persistence using browser local storage
- Responsive design for all devices

## Installation & Usage

### Quick Start (Easiest Way)
1. Download all files from this project
2. Find the file named `index.html`
3. Double-click on `index.html` to open it in your web browser
4. The app will start automatically - no installation needed!


**On Windows:**
- Open Command Prompt (search for "cmd" in Start menu)
- Type: `python -m http.server 8000`
- Open your browser and go to: `http://localhost:8000`

**On Mac or Linux:**
- Open Terminal
- Navigate to the project folder
- Type: `python -m http.server 8000`
- Open your browser and go to: `http://localhost:8000`


## How to Use

1. Enter your name
2. Select an interest area
3. Choose up to three strengths
4. Specify work style preferences
5. View career recommendations
6. Export results or restart
## Technical Details

- Built with HTML5, CSS3, and Vanilla JavaScript
- Uses browser Local Storage for data persistence
- Compatible with Chrome, Firefox, Safari, and Edge
- No backend required - static files only
